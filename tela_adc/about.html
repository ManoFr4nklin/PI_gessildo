Sobre o projeto

O projeto ETE Mix foi criado com o intuito de simular um administrador de um mercado que deseja adicionar produtos em seu mercado, baseado nisso, o projeto foi criado usando HTML,CSS e Javascript, os integrantes que
realizaram esse projeto foram , Thiago Franklin, João Victor Lopes, Filipe Alves , João Victor Nobre, Wallace e Valmir Silva. Além das tecnlogias já citadas, foi usada a distribuição linux Mint para fazer o projeto rodar em uma SO fora do Windows usando uma máquina virtual, além da instalação do servidor web Apache e a utilização de um banco de dados MySQL.
